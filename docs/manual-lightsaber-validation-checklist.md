# lightsaber System Manual Validation Checklist

`dungeon-rpg`가 `lightsaber_test` 문서의 사전 체험 리허설 역할을 하는지
직접 플레이로 확인하기 위한 최소 수동 체크리스트.

---

## 1. 부팅 / 저장 안정성

- [ ] `node scripts/check-boot.js` 통과
- [ ] 빈 localStorage 상태에서 정상 부팅
- [ ] 저장 데이터가 있는 상태에서 정상 부팅
- [ ] 프로필 / 장비 / 수련의 방 / 문장의방 패널이 모두 열린다

---

## 2. 7단 → 문장의방

- [ ] 7단 Lv36 상태를 만들 수 있다
- [ ] 병종 라인을 바꿔 문장의방 진입 조건을 확인할 수 있다
- [ ] 조건이 부족하면 무엇이 부족한지 체감 가능하다
- [ ] 조건 충족 시 시험 전투에 진입 가능하다
- [ ] 시험 승리 후 기본 문장을 획득한다

---

## 3. 문장 장착 의미

- [ ] 획득한 문장을 활성 문장으로 장착할 수 있다
- [ ] 활성 문장을 바꾸면 스탯/체감이 달라진다
- [ ] 머리 슬롯 장비를 끼면 문장과 충돌한다
- [ ] 저장 후 재접속해도 활성 문장 상태가 유지된다

---

## 4. 합체 및 8단 진입

- [ ] 필요한 기본 문장을 모두 모을 수 있다
- [ ] 마스터 문장 융합 버튼이 조건 충족 시 열린다
- [ ] 융합 후 8단 마스터 계열로 바뀐다
- [ ] 프로필 / HUD / 수련의 방에서 계열 변화가 보인다

---

## 5. 9단 / 10단 성장

- [ ] 8단에서 9단 요구 레벨까지 올릴 수 있다
- [ ] 9단 승급이 열린다
- [ ] 10단까지 같은 방식으로 도달 가능하다
- [ ] 10단 클래스명이 최종 계열 이름으로 보인다

---

## 6. 장비 파밍 재미

- [ ] 던전 보스가 후반 장비를 드롭한다
- [ ] 상점이 현재 진행 tier에 맞춰 상위 장비를 판다
- [ ] 더 높은 장비를 얻으면 문장의방 시험과 후반 성장 루프가 체감상 쉬워진다
- [ ] “조금 더 돌아서 장비를 맞추고 싶다”는 감각이 생긴다

---

## 7. 디버그 단축 헬퍼

브라우저 콘솔에서 사용:

- `__rpgDebug.prepareEmblemTrial('infantry')`
- `__rpgDebug.prepareFusion('battle_master_emblem')`
- `__rpgDebug.prepareTier(8, 'battleMaster')`
- `__rpgDebug.prepareTier(9, 'tacticsMaster')`
- `__rpgDebug.prepareTier(10, 'magicMaster')`

이 헬퍼는 검증 속도를 높이기 위한 것이고,
최종 판단은 실제 플레이 루프에서도 성립해야 한다.
