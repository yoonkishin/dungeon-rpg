# Character Fusion V0 Plan

`dungeon-rpg` 프로토타입용 **캐릭터 합체 시스템 초안**.

> 상태: draft only, not implemented
> 기준: `lightsaber_test`의 문장/융합 감각은 참고하되, 현재 `dungeon-rpg` 구조에 맞게 축소 재해석한다.

---

## 1. 왜 지금 합체를 넣는가

현재 `dungeon-rpg`는 이미 아래 축을 갖고 있다.

- 플레이어 성장 / 승급
- 동료 2인 파티
- 동료 AI 모드
- 조합 시너지
- 던전/보스 기믹 1차

이제 필요한 것은
**"이 조합을 오래 키우면 어떤 큰 보상이 열리나"** 에 대한 한 단계 위 목표다.

합체 시스템은 이 문제를 해결한다.

- 승급 이후의 다음 목표 제공
- 동료 조합 선택 이유 강화
- 시너지 시스템을 단순 수치 보너스에서 핵심 전투 이벤트로 승격
- 후반에 `lightsaber_test`식 상위 융합/문장 시스템으로 확장 가능한 다리 역할

---

## 2. 원작 문서와의 관계

`lightsaber_test` 문서 기준 원작 감각은 다음에 가깝다.

- 7단 Lv35 도달
- 문장 획득
- 문장 합체
- 8~10단 마스터/그랑/그랜드 융합 진입

하지만 `dungeon-rpg`는 아직
- 문장 시스템 없음
- 8~10단 장기 성장 없음
- 플레이어 라인도 1개(`infantry`) 중심

그래서 프로토타입에서는 이를 그대로 옮기지 않고,
**"특정 동료 조합을 갖춘 상태에서 전투 중 일시 합체 폼을 발동"** 하는 방식으로 재해석한다.

즉,
원작의 **상위 융합 감각**만 먼저 가져오고,
문장/계승/장기 티어 구조는 나중으로 미룬다.

---

## 3. V0 핵심 컨셉

### 3.1 합체는 "영구 클래스 교체"가 아니다

V0에서 합체는 저장 데이터를 영구적으로 바꾸는 시스템이 아니다.

대신,
- 특정 조합을 갖춘 상태
- 전투 중 조건 충족
- 잠시 동안 강력한 합체 형태 발동

으로 정의한다.

즉 V0는
**궁극기형 합체 시스템**이다.

### 3.2 합체는 시너지의 상위 단계다

현재 시너지:
- 조합 맞추면 패시브 보너스

V0 합체:
- 조합 맞추고 게이지를 채우면
- **액티브 합체 기술** 사용 가능

정리하면:
- 시너지 = 상시 패시브
- 합체 = 조건부 액티브 피니시

---

## 4. 발동 구조

## 4.1 합체 가능 조건

아래 조건을 모두 만족해야 한다.

1. 활성 동료 2명이 특정 합체 조합을 만족한다.
2. 플레이어가 최소 승급 조건을 만족한다. (권장: `classRank >= 3`)
3. 던전 내부 전투 중이다.
4. 합체 게이지가 100%다.
5. 동료 2명이 모두 생존 중이다.
6. 보스전 또는 엘리트전에서 우선 사용 가치가 생기도록 쿨다운이 길다.

## 4.2 합체 게이지 충전 규칙

게이지는 다음 행동으로 누적된다.

- 적 처치
- 보스/엘리트 타격
- 동료 스킬 적중
- 힐/보호 성공
- 시너지 상태 유지 시간

의도:
- 그냥 시간만 지나서 차는 게 아니라
- **조합이 실제로 잘 굴러갈수록 빨리 차야 한다**

## 4.3 발동 결과

합체 발동 시:

- 플레이어와 동료 2명이 잠시 하나의 합체 폼으로 전환
- 동료는 잠시 필드에서 사라짐
- 플레이어 공격/스킬/외형/이동 특성이 강화됨
- 합체 종료 후 동료 복귀

## 5. 추천 합체 계열

현재 구현 구조를 기준으로, V0는 3계열이면 충분하다.

### 5.1 Battle Fusion

조건 예시:
- 보병 + 비병
- 보병 + 기병
- 비병 + 기병

특징:
- 근접 화력 강화
- 돌진/연속 베기/슈퍼아머
- 보스 압박에 강함

표시명 예시:
- 배틀 폼
- 전장 융합
- 전투 지휘 합체

### 5.2 Tactics Fusion

조건 예시:
- 궁병 + 창병
- 수병 + 궁병
- 창병 + 수병

특징:
- 사거리 증가
- 관통/속박/범위 견제
- 엘리트/군중 처리 특화

표시명 예시:
- 택틱스 폼
- 전술 융합

### 5.3 Mystic Fusion

조건 예시:
- 승려 + 신관
- 법사 + 신관
- 법사 + 사교

특징:
- 광역 마법
- 보호막/흡수/폭발
- 안정성과 폭딜 동시 제공

표시명 예시:
- 미스틱 폼
- 마도 융합

## 6. UX 흐름

## 6.1 동료 패널

동료 패널에서 각 조합에 대해 아래를 보여준다.

- 현재 시너지 활성 여부
- 합체 가능 조합 여부
- 잠금 조건
- 합체 계열 설명

예시:
- `배틀 폼 가능: 보병 + 기병`
- `잠금 해제 조건: 플레이어 3단 승급`

## 6.2 HUD

HUD에 추가될 것:

- 합체 게이지 바
- 합체 가능 시 발동 버튼 강조
- 발동 중 남은 시간 표시

예시:
- `합체 72%`
- `합체 가능!`
- `배틀 폼 08.4s`

## 6.3 발동 연출

합체 발동 시 짧게:

- 화면 흔들림
- 색상 오라 변경
- 배너 표시
- 강화된 공격 이펙트

배너 예시:
- `합체! 배틀 폼`
- `합체! 미스틱 폼`

---

## 7. 전투 효과 제안

V0에서는 폼마다 하나의 성격만 분명하면 된다.

### Battle Fusion
- 공격속도 증가
- 근접 범위 증가
- 기본 공격이 2연타화
- 받는 피해 소폭 감소

### Tactics Fusion
- 공격 사거리 증가
- 탄환/충격파 관통
- 적 이동속도 저하
- 다수전 대응 강화

### Mystic Fusion
- 기본 공격에 폭발 판정 추가
- 일정 주기 보호막 또는 소량 회복
- 스킬 피해 강화
- 상태이상/광역 견제 강화

원칙:
- 숫자보다 **조작감 변화**가 먼저 느껴져야 한다.

## 9. 구현 순서 제안

### Phase 1, 감지/표시
- 합체 가능 조합 판정 함수 추가
- 동료 패널에 합체 가능 계열 표시
- HUD에 합체 게이지 표시

### Phase 2, 발동/지속
- 게이지 충전 로직 추가
- 합체 버튼 추가
- 합체 발동 시 플레이어 강화 상태 적용
- 발동 중 동료 숨김 처리

### Phase 3, 폼 개성 강화
- 배틀/택틱스/미스틱별 공격 특성 차등화
- 합체 전용 이펙트/배너 추가
- 보스/엘리트 대응 보너스 조정

---

## 10. 이번 V0에서 하지 않을 것

아래는 일부러 하지 않는다.

- 문장 획득/소모 시스템
- 8~10단 영구 융합 클래스
- 재료 아이템 기반 합체
- 동료 자체 영구 합성/삭제
- 캐릭터 3명 완전 영구 통합
- 복잡한 분기 레시피 다수

즉 V0는
**화려하지만 구현 가능한 수준의 전투형 합체**만 다룬다.

---

## 11. 완료 기준

V0 완료는 아래 기준으로 본다.

1. 특정 조합에서만 합체 가능하다.
2. 합체 게이지가 전투 플레이로 차오른다.
3. 합체 발동 시 플레이 감각이 즉시 달라진다.
4. 합체 종료 후 기존 파티 구조로 안전하게 복귀한다.
5. 현재 save/load와 큰 충돌 없이 유지된다.
6. 이후 문장/상위 융합 시스템으로 확장 가능한 이름과 데이터 구조를 남긴다.

---

## 12. 추천 결론

현재 `dungeon-rpg`에는
원작식 문장 합체 풀버전보다,

**시너지 조합 기반 전투형 합체 V0**

가 먼저 맞다.

이렇게 가면:
- 현재 있는 동료/시너지/보스 구조를 바로 활용할 수 있고
- 구현 부담이 덜하고
- 나중에 `lightsaber_test`식 상위 융합으로 확장할 여지도 남는다.
